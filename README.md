# 豪华微信旅游小程序（weapp-demo）

## 项目说明
这是一个用于展示亚洲高端旅游景点的微信小程序 Demo，包含首页浏览、搜索筛选、详情展示与收藏功能。

## 运行方式
- 使用微信开发者工具打开本目录即可运行
- 推荐基础库版本：2.31+

## 功能概览
1. 首页
   - 景点列表展示
   - 支持关键词搜索
   - 国家筛选、标签筛选
   - 下拉刷新、分页加载

2. 详情页
   - 景点封面与图集浏览
   - 景点简介与详细说明
   - 收藏 / 取消收藏
   - 分享功能

3. 收藏页
   - 本地收藏数据存储
   - 可跳转详情或取消收藏

## 数据说明
- 景点数据位于：`data/attractions.js`
- 每个景点包含：
  - id
  - name
  - country
  - tags
  - cover
  - gallery
  - summary
  - detail

## 资源配置
- 静态资源基路径通过 `config.js` 中的 ASSET_BASE 配置
- 可对接 CDN 或 VPS 静态资源服务

## 说明
本项目为 Demo 示例，可在此基础上继续扩展页面样式、真实数据接口与业务逻辑。
